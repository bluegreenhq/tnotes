# tnotes ユーザーマニュアル

## 概要

tnotes はターミナル上で動作するメモ管理アプリケーションです。TUI モードと CLI モードの 2 つの使い方があります。

- **TUI モード**: Mac 標準の Notes アプリに近い 2 ペインレイアウトで、左にノートリスト、右にテキストエディタを表示します。マウスとキーボードの両方で操作できます。
- **CLI モード**: サブコマンドでノートの一覧表示や内容の取得ができます。スクリプトやパイプラインとの連携に便利です。

## CLI モード

引数なしで起動すると TUI モードになります。サブコマンドを指定すると CLI モードで動作します。

```bash
tnotes              # TUIモードで起動
tnotes list         # ノート一覧を表示
tnotes get <id>     # 指定IDのノートを表示
tnotes create [file] # ファイルまたは標準入力からノートを作成
tnotes export <file> # データ一式をzipにエクスポート
tnotes import <file> # zipからデータをインポート
tnotes help         # ヘルプを表示
```

### list

ノートの ID、タイトル、更新日時を一覧表示します。

```
$ tnotes list
ID                TITLE          UPDATED
db7b7ce542656e1a  Hello, Notes!  2026-04-04 11:40
8a2816cfb412cd21  Meeting Notes  2026-04-04 11:33
```

### get

指定した ID のノート本文を標準出力に出力します。`list` で確認した ID を指定します。

```
$ tnotes get db7b7ce542656e1a
Hello, Notes!
This is a TUI app.
```

### create

ファイルまたは標準入力からノートを作成します。作成されたノートの ID が出力されます。

```
# ファイルから作成
$ tnotes create memo.txt
a1b2c3d4e5f67890

# 標準入力から作成（パイプ）
$ echo "Meeting memo" | tnotes create
b2c3d4e5f6789012

# ヒアドキュメントから作成
$ tnotes create <<EOF
Shopping List
- Milk
- Eggs
EOF
c3d4e5f67890abcd
```

1 行目がノートのタイトルとして扱われます。入力が空の場合はエラーになります。

### export

データディレクトリの内容（通常ノート、ゴミ箱含む）をzipファイルにエクスポートします。ディレクトリ構造がそのまま保持されます。

```
$ tnotes export backup.zip
Exported to backup.zip
```

出力先に既存ファイルがある場合はエラーになります。

### import

zipファイルからデータディレクトリにインポートします。既存データがある（`index.json` が存在する）場合はエラーになります。

```
# 新しい環境への復元
$ NOTES_DATA_DIR=/tmp/new-notes tnotes import backup.zip
Imported from backup.zip
```

export と import を組み合わせることで、データのバックアップや別環境への移行ができます。

## 操作方法

### キーボード

| キー | コンテキスト | 動作 |
|------|-------------|------|
| `n` | サイドバー | 新しいノートを作成しエディタへ移動 |
| `d` / `Delete` / `Backspace` | サイドバー | 選択中ノートをゴミ箱へ移動 |
| `g` | サイドバー | ゴミ箱モード ↔ 通常モードの切り替え |
| `r` | ゴミ箱モード | 選択中ノートを復元 |
| `j` / `↓` / `Ctrl+N` | サイドバー | 次のノートを選択 |
| `k` / `↑` / `Ctrl+P` | サイドバー | 前のノートを選択 |
| `Enter` | サイドバー | エディタへフォーカス移動 |
| `Tab` | どこでも | サイドバー ↔ エディタのフォーカス切り替え |
| `Ctrl+Z` | サイドバー | ノート操作の取り消し（作成・削除・復元） |
| `Ctrl+Shift+Z` | サイドバー | ノート操作のやり直し |
| `Ctrl+Z` | エディタ | テキスト編集の取り消し |
| `Ctrl+Shift+Z` | エディタ | テキスト編集のやり直し |
| `Ctrl+Shift+A` | エディタ | テキスト全選択 |
| `Ctrl+A` | エディタ | 行頭に移動 |
| `Ctrl+E` | エディタ | 行末に移動 |
| `Ctrl+F` | エディタ | 右に1文字移動 |
| `Ctrl+B` | エディタ | 左に1文字移動 |
| `Ctrl+N` | エディタ | 下に1行移動 |
| `Ctrl+P` | エディタ | 上に1行移動 |
| `Ctrl+D` | エディタ | カーソル位置の文字を削除 |
| `Ctrl+K` | エディタ | カーソルから行末まで削除 |
| `Ctrl+C` | エディタ | 選択テキストをクリップボードにコピー |
| `Ctrl+X` | エディタ | 選択テキストをクリップボードにコピーし削除 |
| `Ctrl+V` | エディタ | クリップボードからテキストをペースト |
| `Shift+←` / `Shift+→` | エディタ | テキストを1文字単位で範囲選択 |
| `Shift+↑` / `Shift+↓` | エディタ | テキストを1行単位で範囲選択 |
| `Esc` | エディタ（選択中） | 選択解除 |
| `Esc` | エディタ（選択なし） | サイドバーへフォーカス移動 |
| `q` | サイドバー | アプリケーション終了 |

### マウス

| 操作 | 動作 |
|------|------|
| ノートリストをクリック | 該当ノートを選択 |
| エディタ領域をクリック | エディタへフォーカス移動、カーソル移動 |
| エディタ領域をドラッグ | テキストを範囲選択 |
| `[New]` クリック | 新しいノートを作成 |
| `[Trash]` クリック | ゴミ箱モードに切り替え |
| `[Notes]` クリック | 通常モードに切り替え（ゴミ箱モード時） |
| `[Restore]` クリック | 選択中ノートを復元（ゴミ箱モード時） |
| `[Copy]` クリック | 選択テキストをクリップボードにコピー（選択時のみ表示） |
| `[Cut]` クリック | 選択テキストをクリップボードにコピーし削除（選択時のみ表示） |
| `[Quit]` クリック | アプリケーション終了 |

フッターのボタンはマウスホバー時に下線でハイライトされます。テキスト選択中は `[Copy]` と `[Cut]` ボタンがフッター右側に追加表示されます。

## ノート管理

### ノートを作成する

1. `n` キーまたは `[New]` ボタンでノートを作成
2. エディタに自動的にフォーカスが移動する
3. テキストを入力する（1 行目がタイトルになる）

### ノートを編集する

1. サイドバーでノートを選択（`j`/`k` またはクリック）
2. `Tab`、`Enter`、またはエディタ領域クリックでエディタへ移動
3. テキストを編集する
4. `Esc` でサイドバーに戻ると変更が保存される

### テキストを選択・コピー・カット・ペーストする

1. エディタ内でマウスドラッグ、`Shift+矢印キー` で範囲選択、または `Ctrl+Shift+A` でテキスト全選択する
2. 選択範囲は青背景でハイライト表示される
3. `Ctrl+C` またはフッターの `[Copy]` で選択テキストをシステムクリップボードにコピー
4. `Ctrl+X` またはフッターの `[Cut]` で選択テキストをクリップボードにコピーしつつ削除
5. `Ctrl+V` でクリップボードからテキストをペースト（選択範囲がある場合は置換）
6. 選択中に文字を入力すると、選択範囲が入力テキストに置換される
7. `Esc`、別の場所をクリック、または矢印キー（Shift なし）で選択が解除される

### ノートを削除する

1. サイドバーでノートを選択
2. `d`、`Delete`、`Backspace` のいずれか、または `[Trash]` ボタンでゴミ箱に移動
3. 確認なしで即座にゴミ箱に移動される

### ゴミ箱からノートを復元する

1. `g` キーまたは `[Trash]` ボタンでゴミ箱モードに切り替え
2. サイドバーに削除済みノートが一覧表示される
3. エディタにノート内容が読み取り専用で表示される
4. `r` キーまたは `[Restore]` ボタンで選択中ノートを復元
5. 復元後は自動的に通常モードに戻り、復元したノートが選択される
6. `g` キーまたは `[Notes]` ボタンで通常モードに戻る

### 操作の取り消し・やり直し（Undo/Redo）

ノート操作（作成・削除・復元）とテキスト編集の両方で Undo/Redo が使えます。

- **サイドバー**: `Ctrl+Z` でノート操作（作成・ゴミ箱移動・復元）を取り消し、`Ctrl+Shift+Z` でやり直し
- **エディタ**: `Ctrl+Z` でテキスト編集を取り消し、`Ctrl+Shift+Z` でやり直し

操作後にフッターに「Undo: Ctrl+Z」などのヒントが表示されます。

### ノートの並び順

ノートは更新日時の降順（新しいものが上）で自動的に並び替えられます。

## トラブルシューティング

### 「Terminal too small」と表示される

ターミナルウィンドウを最低 80 カラム以上に広げてください。

### フッターのボタンが表示されない

ターミナルの高さが不足しています。ウィンドウを縦に広げてください。

### エディタでの編集内容が反映されない

`Esc` または `Tab` でサイドバーに戻ると、編集内容がノートに保存されます。アプリ終了時（`q`、`Ctrl+C`、`[Quit]`）にも自動保存されます。

## データの保存

### 保存場所

ノートは以下の優先順位で決定されるディレクトリに保存されます。

1. `$NOTES_DATA_DIR`（環境変数で直接指定）
2. `$XDG_DATA_HOME/tnotes/`（XDG Base Directory 仕様）
3. `~/.local/share/tnotes/`（デフォルト）

### ファイル構成

```
~/.local/share/tnotes/
├── index.json          # メタデータインデックス（起動時に使用）
├── YYYYMMDD/           # 作成日ごとのディレクトリ
│   └── {id}.md         # フロントマター付きMarkdownファイル
└── .trash/             # ゴミ箱ディレクトリ
    ├── index.json      # ゴミ箱メタデータインデックス
    └── YYYYMMDD/
        └── {id}.md     # 削除されたノートファイル
```

各ノートファイルはフロントマター付きの Markdown 形式で保存されるため、外部のエディタやツールからも直接読み書きできます。

### 保存タイミング

以下のタイミングで自動的に保存されます。

- エディタからサイドバーへフォーカス移動時（`Esc`、`Tab`）
- 別のノートを選択した時
- アプリケーション終了時（`q`、`[Quit]`）

## 制限事項

- ゴミ箱からの完全削除（パージ）機能は未実装です
- ソフトラップ（行の折り返し）は未対応です。長い行は画面幅で切り捨てて表示されます
